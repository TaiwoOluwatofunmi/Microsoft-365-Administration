## 🚀 Project Description
This project showcases hands-on experience in administering Microsoft 365 environments, focusing on **group management, user access control, and enterprise communication workflows**.

I simulated a real-world IT support scenario by creating and configuring a Microsoft 365 Group, managing users, and validating internal communication using Outlook.  

The project demonstrates practical skills in:
- Identity and access management  
- Microsoft 365 administration  
- Troubleshooting email delivery issues  
- Supporting collaboration tools used in modern organizations  

This aligns with responsibilities typically expected in **IT Support, Cloud Support, and Microsoft 365 Administrator roles**.
