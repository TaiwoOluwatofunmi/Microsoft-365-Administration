## 👤 User Account Management – Creation & Password Reset

### Scenario
A new employee (Paul Silas) was onboarded into the organization and later experienced login issues due to a forgotten password.

### Actions Taken
- Created a new user account in Microsoft 365 Admin Center
- Assigned appropriate license to enable services
- Provided initial login credentials
- Reset user password after login issue
- Tested login to confirm successful access

### Outcome
User account was successfully created and access restored after password reset.

### Tools Used
- Microsoft 365 Admin Center
